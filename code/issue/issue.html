# 前言

服务器解压乱码问题

# 1.unzip解压中文文件名乱码

unzip解压中文文件名乱码

# 问题分析

由于中文的Windows使用的是GBK编码

而Linux默认使用UTF-8编码的

如果在Windows打包带中文文件的zip包或打包后的名字带中文

会乱码

# 解决方法

通过命令处理：

unzip -O GBK *.zip

# 2.密码自动填充问题

# 问题分析

由于谷歌的自动填充导致修改用户密码的时候密码会自动填充为登录密码

# 解决方法

设置属性

autocomplete="new-password"
